# Setting Up

[![Build Status](https://travis-ci.org/sks/linuxkit.svg?branch=master)](https://travis-ci.org/sks/linuxkit)

## Dependencies

- Install [linuxkit](https://github.com/linuxkit/linuxkit)

```sh
brew tap linuxkit/linuxkit
brew install --HEAD linuxkit
```

## Create linux

```sh
make run
```
